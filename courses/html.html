1. Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the meaning and structure of web content.

What is HTML?

HTML is a markup language that uses tags to define elements within a document. These elements form the building blocks of web pages.

Basic HTML Document

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

2. HTML Page Structure

Every HTML document has a specific structure that tells the browser how to display the content.

Document Type Declaration

<!DOCTYPE html>

This declaration defines the document type and version of HTML.

HTML Root Element

<html>
    <!-- Page content goes here -->
</html>

The <html> element is the root element of an HTML page.

Head Section

<head>
    <title>Page Title</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

The <head> element contains meta information about the document.

Body Section

<body>
    <h1>Welcome to My Website</h1>
    <p>This is the visible content.</p>
</body>

The <body> element contains the visible page content.

Complete Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>Hello, world!</p>
</body>
</html>

7. Semantic Tags

Use semantic HTML tags to create meaningful and accessible web pages.

What are Semantic Tags?

Semantic tags clearly describe their meaning in a human- and machine-readable way. They help search engines and assistive technologies understand the content structure.

Common Semantic Elements

<header>
    <!-- Page or section header content -->
</header>

<nav>
    <!-- Navigation links -->
</nav>

<main>
    <!-- Main content of the page -->
</main>

<article>
    <!-- Independent, self-contained content -->
</article>

<section>
    <!-- Thematic grouping of content -->
</section>

<aside>
    <!-- Content tangentially related to the main content -->
</aside>

<footer>
    <!-- Page or section footer content -->
</footer>

Complete Semantic Page Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Semantic HTML Example</title>
</head>
<body>
    <header>
        <h1>My Website</h1>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <header>
                <h2>Article Title</h2>
                <p>Published on <time datetime="2023-01-01">January 1, 2023</time></p>
            </header>
            <p>Article content goes here...</p>
        </article>
        
        <section>
            <h2>Related Content</h2>
            <p>More content in this section...</p>
        </section>
    </main>
    
    <aside>
        <h3>Sidebar</h3>
        <p>Additional information...</p>
    </aside>
    
    <footer>
        <p>&copy; 2023 My Website. All rights reserved.</p>
    </footer>
</body>
</html>

8. Audio & Video Embedding

Learn how to embed audio and video content in your web pages.

Embedding Audio

<!-- Basic audio player -->
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>

<!-- Audio with additional attributes -->
<audio controls autoplay muted loop>
    <source src="audio.mp3" type="audio/mpeg">
    <a href="audio.mp3">Download audio</a>
</audio>

Embedding Video

<!-- Basic video player -->
<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Your browser does not support the video tag.
</video>

<!-- Video with poster and additional attributes -->
<video width="640" height="480" controls poster="thumbnail.jpg" autoplay muted loop>
    <source src="video.mp4" type="video/mp4">
    <track src="captions.vtt" kind="subtitles" srclang="en" label="English">
    Your browser does not support the video tag.
</video>

9. Iframes and Embedding External Content

Embed external content like maps, videos, and other web pages using iframes.

What is an iframe?

An iframe (inline frame) is an HTML element that allows you to embed another HTML document within the current document.

Basic iframe Usage

<!-- Basic iframe -->
<iframe src="https://www.example.com"></iframe>

<!-- iframe with dimensions -->
<iframe src="https://www.example.com" width="600" height="400"></iframe>

<!-- iframe with additional attributes -->
<iframe src="https://www.example.com" 
        width="600" 
        height="400" 
        frameborder="0" 
        allowfullscreen>
</iframe>

Embedding YouTube Videos

<iframe width="560" 
        height="315" 
        src="https://www.youtube.com/embed/VIDEO_ID" 
        title="YouTube video player" 
        frameborder="0" 
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
        allowfullscreen>
</iframe>

10. HTML5 APIs

Explore powerful HTML5 APIs that enable advanced functionality in web applications.

Canvas API

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
    const canvas = document.getElementById('myCanvas');
    const ctx = canvas.getContext('2d');
    
    // Draw a rectangle
    ctx.fillStyle = '#FF0000';
    ctx.fillRect(0, 0, 150, 75);
</script>

Geolocation API

<button onclick="getLocation()">Get Location</button>
<p id="location"></p>

<script>
    function getLocation() {
        if (navigator.geolocation) {
            navigator.geolocation.getCurrentPosition(showPosition);
        } else {
            document.getElementById("location").innerHTML = "Geolocation is not supported by this browser.";
        }
    }
    
    function showPosition(position) {
        document.getElementById("location").innerHTML = 
            "Latitude: " + position.coords.latitude + 
            "<br>Longitude: " + position.coords.longitude;
    }
</script>

Web Storage API

<script>
    // Local Storage
    localStorage.setItem('username', 'JohnDoe');
    const username = localStorage.getItem('username');
    localStorage.removeItem('username');
    
    // Session Storage
    sessionStorage.setItem('theme', 'dark');
    const theme = sessionStorage.getItem('theme');
    sessionStorage.removeItem('theme');
</script>
