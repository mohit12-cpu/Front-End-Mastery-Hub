1. Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It emphasizes code readability and allows programmers to express concepts in fewer lines of code than languages like C++ or Java.

What is Python?

Python is a general-purpose programming language that can be used for:

  • Web development (Django, Flask)
  • Data science and machine learning (Pandas, NumPy, TensorFlow)
  • Automation and scripting
  • Desktop applications (Tkinter)
  • Game development (Pygame)

Python Features

  • Easy to learn - Simple syntax similar to English
  • Interpreted - No need to compile code before execution
  • Cross-platform - Works on Windows, Mac, Linux
  • Extensive libraries - Large standard library and third-party packages
  • Object-oriented - Supports OOP concepts

Sample Python Code

2. Python Syntax and Variables

Python has a clean and readable syntax that makes it easy to write and understand code.

Python Syntax Rules

  • Case-sensitive - Variables name and Name are different

4. Control Flow

Control flow statements allow you to control the execution flow of your Python programs based on certain conditions.

Conditional Statements

# If statement
x = 10
if x > 5:
    print("x is greater than 5")

# If-else statement

    print("y is even")

    print("y is odd")

score = 85
if score >= 90:
    grade = "A"

    grade = "B"

    grade = "C"

    grade = "D"

num = 15
if num > 0:
    if num % 2 == 0:

    else:

5. Loops

Loops allow you to execute a block of code repeatedly, which is essential for processing collections of data and automating repetitive tasks.

While Loops

# Basic while loop
count = 0
while count < 5:
    print(f"Count is {count}")
    count += 1

# While loop with user input
user_input = ""
while user_input != "quit":
    user_input = input("Enter 'quit' to exit: ")
    print(f"You entered: {user_input}")

# Infinite loop with break
while True:
    number = int(input("Enter a number (0 to quit): "))
    if number == 0:
        break
    print(f"Square of {number} is {number**2}")

# Continue statement
num = 0
while num < 10:
    num += 1
    if num % 2 == 0:
        continue  # Skip even numbers
    print(num)  # Print odd numbers

For Loops

# For loop with range
for i in range(5):
    print(i)  # Prints 0, 1, 2, 3, 4

# For loop with start, stop, step
for i in range(2, 10, 2):
    print(i)  # Prints 2, 4, 6, 8

# For loop with lists
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(f"I like {fruit}")

# For loop with strings
word = "Python"
for letter in word:
    print(letter)

# For loop with enumerate
fruits = ["apple", "banana", "orange"]
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

# For loop with dictionaries
person = {"name": "Alice", "age": 30, "city": "New York"}
for key, value in person.items():
    print(f"{key}: {value}")

Nested Loops

# Nested for loops

# Pattern printing

# Multiplication table

8. File Handling

File handling in Python allows you to read from and write to files, which is essential for data persistence and working with external data sources.

Reading Files

# Reading entire file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

# Reading line by line
with open("example.txt", "r") as file:
    for line in file:
        print(line.strip())  # strip() removes newline characters

# Reading all lines into a list
with open("example.txt", "r") as file:
    lines = file.readlines()
    print(lines)

# Reading with error handling
try:
    with open("nonexistent.txt", "r") as file:
        content = file.read()
except FileNotFoundError:
    print("File not found!")

Writing Files

# Writing to a file (overwrites existing content)
with open("output.txt", "w") as file:

9. Exception Handling

Exception handling allows you to gracefully handle errors and unexpected situations in your Python programs, preventing crashes and providing better user experience.

Basic Exception Handling

# Basic try-except
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

# Multiple exceptions
try:
    number = int(input("Enter a number: "))
    result = 10 / number
except ValueError:
    print("Invalid input! Please enter a number.")
except ZeroDivisionError:
    print("Cannot divide by zero!")

# Generic exception
try:
    # Some risky operation
    pass
except Exception as e:
    print(f"An error occurred: {e}")

# Else and finally clauses
try:
    number = int(input("Enter a number: "))
except ValueError:
    print("Invalid input!")
else:
    print(f"You entered: {number}")  # Executes if no exception
finally:
    print("This always executes")  # Always executes

Raising Exceptions

# Raising built-in exceptions
def validate_age(age):
    if age < 0:
        raise ValueError("Age cannot be negative")
    if age > 150:
        raise ValueError("Age seems unrealistic")
    return True

try:
    validate_age(-5)
except ValueError as e:
    print(f"Validation error: {e}")

# Creating custom exceptions
class CustomError(Exception):
    pass

class AgeValidationError(Exception):
    def __init__(self, age, message="Invalid age"):
        self.age = age
        self.message = message
        super().__init__(self.message)
    
    def __str__(self):
        return f"{self.message}: {self.age}"

# Using custom exceptions
try:
    age = -5
    if age < 0:
        raise AgeValidationError(age)
except AgeValidationError as e:
    print(e)

Exception Best Practices

# Specific exceptions first
try:
    # Some code
    pass
except ValueError:  # More specific
    print("Value error")
except Exception:   # More general
    print("General error")

# Don't catch exceptions you can't handle
def process_data(data):
    try:
        return int(data) * 2
    except ValueError:
        # Log the error and re-raise
        print(f"Cannot convert {data} to integer")
        raise  # Re-raise the same exception

# Context managers for resource cleanup
# Instead of:
file = open("data.txt")
try:
    content = file.read()
finally:
    file.close()

# Use:
with open("data.txt") as file:
    content = file.read()
# File automatically closed

10. Object-Oriented Programming

Object-Oriented Programming (OOP) is a programming paradigm that organizes code around objects, which are instances of classes. OOP helps create more modular, reusable, and maintainable code.

Classes and Objects

# Defining a class
class Person:
    # Class attribute
    species = "Homo sapiens"
    
    # Constructor
    def __init__(self, name, age):
        # Instance attributes
        self.name = name
        self.age = age
    
    # Instance method
    def introduce(self):
        return f"Hi, I'm {self.name} and I'm {self.age} years old."
    
    # Another instance method
    def have_birthday(self):
        self.age += 1
        return f"Happy birthday! I'm now {self.age} years old."

# Creating objects (instances)
person1 = Person("Alice", 25)
person2 = Person("Bob", 30)

# Accessing attributes and methods
print(person1.name)  # Alice
print(person2.introduce())  # Hi, I'm Bob and I'm 30 years old.
print(Person.species)  # Homo sapiens

# Modifying attributes
person1.have_birthday()  # Happy birthday! I'm now 26 years old.

Encapsulation

# Private attributes and methods
class BankAccount:
    def __init__(self, owner, initial_balance=0):
        self.owner = owner
        self._balance = initial_balance  # Protected attribute
        self.__account_number = self._generate_account_number()  # Private attribute
    
    def _generate_account_number(self):  # Protected method
        import random
        return random.randint(100000, 999999)
    
    def __validate_amount(self, amount):  # Private method
        return amount > 0
    
    def deposit(self, amount):
        if self.__validate_amount(amount):
            self._balance += amount
            return f"Deposited ${amount}. New balance: ${self._balance}"
        else:
            return "Invalid amount"
    
    def withdraw(self, amount):
        if self.__validate_amount(amount) and amount <= self._balance:
            self._balance -= amount
            return f"Withdrew ${amount}. New balance: ${self._balance}"
        else:
            return "Invalid amount or insufficient funds"
    
    def get_balance(self):
        return self._balance
    
    # Property decorator for controlled access
    @property
    def account_number(self):
        return self.__account_number
    
    @property
    def balance(self):
        return self._balance

# Using the BankAccount class
account = BankAccount("Alice", 1000)
print(account.deposit(500))  # Deposited $500. New balance: $1500
print(account.withdraw(200))  # Withdrew $200. New balance: $1300
print(account.balance)  # 1300
# print(account.__account_number)  # This would cause an AttributeError

Inheritance

# Base class
class Animal:
    def __init__(self, name, species):
        self.name = name
        self.species = species
    
    def make_sound(self):
        return "Some generic sound"
    
    def info(self):
        return f"{self.name} is a {self.species}"

# Derived classes
class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name, "Dog")
        self.breed = breed
    
    # Method overriding
    def make_sound(self):
        return "Woof!"
    
    # New method
    def fetch(self):
        return f"{self.name} is fetching the ball!"

class Cat(Animal):
    def __init__(self, name, color):
        super().__init__(name, "Cat")
        self.color = color
    
    # Method overriding
    def make_sound(self):
        return "Meow!"
    
    # New method
    def climb(self):
        return f"{self.name} is climbing a tree!"

# Using inherited classes
dog = Dog("Buddy", "Golden Retriever")
cat = Cat("Whiskers", "Orange")

print(dog.info())  # Buddy is a Dog
print(dog.make_sound())  # Woof!
print(dog.fetch())  # Buddy is fetching the ball!

print(cat.info())  # Whiskers is a Cat
print(cat.make_sound())  # Meow!
print(cat.climb())  # Whiskers is climbing a tree!

Polymorphism

# Polymorphism example
class Shape:
    def area(self):
        pass
    
    def perimeter(self):
        pass

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    
    def area(self):
        return self.width * self.height
    
    def perimeter(self):
        return 2 * (self.width + self.height)

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    
    def area(self):
        import math
        return math.pi * self.radius ** 2
    
    def perimeter(self):
        import math
        return 2 * math.pi * self.radius

# Polymorphism in action
shapes = [
    Rectangle(5, 3),
    Circle(4),
    Rectangle(2, 8),
    Circle(3)
]

for shape in shapes:
    print(f"Area: {shape.area():.2f}, Perimeter: {shape.perimeter():.2f}")

12. Mini Project: Guessing Game

In this lesson, we'll build a complete interactive guessing game using all the Python concepts we've learned. This project will incorporate functions, data structures, file handling, exception handling, and object-oriented programming.

Project Overview

We'll create a number guessing game with the following features:

  • Random number generation
  • User input validation
  • Attempt tracking
  • Score calculation
  • High score persistence using file handling
  • Multiple difficulty levels
  • Game statistics

Complete Project Code

import random
import json
import os
from datetime import datetime

class GuessingGame:
    def __init__(self):
        self.high_scores_file = "high_scores.json"
        self.high_scores = self.load_high_scores()
        self.current_score = 0
        self.games_played = 0
        self.games_won = 0
    
    def load_high_scores(self):
        """Load high scores from file."""
        if os.path.exists(self.high_scores_file):
            try:
                with open(self.high_scores_file, 'r') as file:
                    return json.load(file)
            except (json.JSONDecodeError, IOError):
                return {}
        return {}
    
    def save_high_scores(self):
        """Save high scores to file."""
        try:
            with open(self.high_scores_file, 'w') as file:
                json.dump(self.high_scores, file, indent=2)
        except IOError:
            print("Could not save high scores to file.")
    
    def get_high_score(self, difficulty):
        """Get high score for a difficulty level."""
        return self.high_scores.get(difficulty, 0)
    
    def update_high_score(self, difficulty, score):
        """Update high score if current score is better."""
        current_high = self.get_high_score(difficulty)
        if score > current_high:
            self.high_scores[difficulty] = score
            self.save_high_scores()
            return True
        return False
    
    def select_difficulty(self):
        """Let player select difficulty level."""
        print("\nSelect Difficulty:")
        print("1. Easy (1-50, 10 attempts)")
        print("2. Medium (1-100, 7 attempts)")
        print("3. Hard (1-200, 5 attempts)")
        
        while True:
            try:
                choice = int(input("Enter choice (1-3): "))
                if choice == 1:
                    return 50, 10, "Easy"
                elif choice == 2:
                    return 100, 7, "Medium"
                elif choice == 3:
                    return 200, 5, "Hard"
                else:
                    print("Please enter 1, 2, or 3.")
            except ValueError:
                print("Please enter a valid number.")
    
    def play_round(self):
        """Play a single round of the game."""
        self.games_played += 1
        max_number, max_attempts, difficulty = self.select_difficulty()
        
        # Generate random number
        secret_number = random.randint(1, max_number)
        attempts = 0
        guessed_correctly = False
        
        print(f"\nI'm thinking of a number between 1 and {max_number}")
        print(f"You have {max_attempts} attempts to guess it!")
        print(f"Current High Score ({difficulty}): {self.get_high_score(difficulty)}")
        
        while attempts < max_attempts and not guessed_correctly:
            try:
                guess = int(input(f"\nAttempt {attempts + 1}/{max_attempts}. Enter your guess: "))
                
                if guess < 1 or guess > max_number:
                    print(f"Please enter a number between 1 and {max_number}")
                    continue
                
                attempts += 1
                
                if guess == secret_number:
                    guessed_correctly = True
                    self.games_won += 1
                    self.current_score = (max_attempts - attempts + 1) * 10
                    
                    print(f"\n🎉 Congratulations! You guessed it!")
                    print(f"The number was {secret_number}")
                    print(f"It took you {attempts} attempts")
                    print(f"Your score: {self.current_score}")
                    
                    # Update high score
                    if self.update_high_score(difficulty, self.current_score):
                        print("🏆 New high score!")
                    
                elif guess < secret_number:
                    print("Too low! Try a higher number.")
                else:
                    print("Too high! Try a lower number.")
                    
            except ValueError:
                print("Please enter a valid number.")
        
        if not guessed_correctly:
            print(f"\n😞 Game Over! The number was {secret_number}")
            self.current_score = 0
        
        return guessed_correctly
    
    def show_statistics(self):
        """Show game statistics."""
        if self.games_played == 0:
            print("No games played yet!")
            return
        
        win_rate = (self.games_won / self.games_played) * 100
        print(f"\n📊 Game Statistics:")
        print(f"Games Played: {self.games_played}")
        print(f"Games Won: {self.games_won}")
        print(f"Win Rate: {win_rate:.1f}%")
        
        print(f"\n🏆 High Scores:")
        for difficulty, score in self.high_scores.items():
            print(f"{difficulty}: {score}")
    
    def play(self):
        """Main game loop."""
        print("🎯 Welcome to the Number Guessing Game!")
        
        while True:
            print("\n" + "="*40)
            print("1. Play Game")
            print("2. View Statistics")
            print("3. Quit")
            
            try:
                choice = int(input("Enter your choice (1-3): "))
                
                if choice == 1:
                    self.play_round()
                elif choice == 2:
                    self.show_statistics()
                elif choice == 3:
                    print("Thanks for playing! Goodbye! 👋")
                    break
                else:
                    print("Please enter 1, 2, or 3.")
                    
            except ValueError:
                print("Please enter a valid number.")

# Run the game
if __name__ == "__main__":
    game = GuessingGame()
    game.play()
