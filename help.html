Getting Started

How do I create an account?

CodeMaster is a completely free platform that works in your browser. No account creation is required to start learning. All your progress is automatically saved in your browser's local storage.

How do I start learning?

Visit the Courses page to browse available courses. Click on any course to begin learning. Each course is divided into lessons that you can complete at your own pace.

Is there a mobile app?

CodeMaster works on all devices with a modern web browser, including smartphones and tablets. There is currently no dedicated mobile app, but the website is fully responsive and mobile-friendly.

Courses

What courses are available?

CodeMaster currently offers courses in HTML, CSS, JavaScript, and Python. Each course includes interactive tutorials, coding exercises, and quizzes to test your knowledge.

How are courses structured?

Each course is divided into lessons that build upon each other. Lessons include explanations, examples, and interactive coding exercises. Courses conclude with a final project and quiz.

Can I skip lessons?

Yes, you can navigate to any lesson in a course. However, we recommend following the structured order for the best learning experience.

Tutorials

How do the interactive tutorials work?

Each tutorial includes a code editor where you can write and test code in real-time. The output is displayed immediately, allowing you to see the results of your code.

Can I save my code?

Your code is automatically saved in your browser's local storage as you type. When you return to a lesson, your previous code will be restored.

What programming languages are supported?

The tutorials support HTML, CSS, and JavaScript. Python tutorials display code examples but do not include an interactive editor due to browser limitations.

Quizzes

How do quizzes work?

Each course concludes with a quiz to test your knowledge. Quizzes consist of multiple-choice questions. Your scores are saved and can be viewed in your dashboard.

Can I retake quizzes?

Yes, you can retake any quiz as many times as you like. Only your highest score is saved.

What happens if I fail a quiz?

There's no passing or failing - quizzes are designed to help you identify areas where you may need additional study. Review the course material and try again.

Progress Tracking

How is my progress tracked?

Your progress is automatically saved in your browser's local storage. You can view your progress on the Dashboard and Profile pages.

Can I reset my progress?

Yes, you can reset your progress from the Profile page. This will clear all saved data, including completed lessons and quiz scores.

Is my progress synced across devices?

Progress is stored locally in your browser. To sync across devices, you would need to use the same browser and ensure local storage is not cleared.

Troubleshooting

My progress isn't saving

Ensure your browser is not blocking local storage. Try refreshing the page or clearing your browser cache. If issues persist, try a different browser.

The code editor isn't working

Make sure you're using a modern browser (Chrome, Firefox, Safari, or Edge). Try refreshing the page or clearing your browser cache.

Some features aren't loading

Check your internet connection and ensure JavaScript is enabled in your browser. Try refreshing the page or clearing your browser cache.
